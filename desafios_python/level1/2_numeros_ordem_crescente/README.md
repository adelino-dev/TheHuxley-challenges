# 2 números em ordem crescente

_Tempo máximo de execução: 1s_

_Tópicos: decisão, ordenação_

## Descrição

Receba dois números inteiros e os imprima em ordem crescente.

## Formato de entrada

Dois números inteiros separados por um espaço em branco.

## Formato de saída

Dois números inteiros separados por um espaço em branco.

## Exemplos de:

### Entrada

    3 2

### Saída

    2 3
