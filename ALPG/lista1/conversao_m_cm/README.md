# Conversão de m para cm.

## Descrição
Faça um programa em Java que leia do usuário um valor em metros converta para centímetros.

## Formato de entrada

Valor em metros.

## Formato de saída

Valor em centímetros.

## Exemplos de:

| Entrada | Saída |
| -- | -- |
| 50 | Insira um valor em metros: |
|  | Resultado: 5000.0 |
