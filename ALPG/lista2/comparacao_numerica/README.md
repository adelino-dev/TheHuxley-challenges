# Comparação numérica

## Descrição
Faça um programa em Java que peça dois números e imprima o maior deles.

## Formato de entrada

Dois números distintos.

## Formato de saída

O maior número.

## Exemplos de:
___________________________________________
### Entrada
    6
    9

### Saída
    Digite um numero:
    Digite outro numero:
    Maior numero: 9.0
